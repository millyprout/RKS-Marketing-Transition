Accounts, Permissions & Ownership

Because Helena is already logged in, this is an access-validation exercise rather than an account-creation exercise.

Security rule

Do not store passwords inside this onboarding site, Notion pages, platform directories, or shared documents. Record ownership and recovery procedures only in the approved secure system.

Working Standards

File naming

Use a consistent pattern such as Brand_Project_Asset_YYYY-MM-DD_v01.

Versions

Keep working versions separate from approved finals. Avoid names such as “final-final-2.”

Storage

Save editable sources, approved finals, and exports in their designated shared folders.

Links

Link Notion projects to the source file or folder rather than uploading duplicate assets.

Handoffs

Every handoff should identify the next owner, requested action, due date, and relevant links.

Archiving

Move obsolete or superseded work to the archive without deleting useful historical context.
