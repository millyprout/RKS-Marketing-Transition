Access & Security Checklist

Do not place passwords inside this directory or in shared documents.

  • Confirm Helena has the correct role, not only a shared login.
  • Record the account owner, recovery email, and two-factor authentication process in the approved secure system.
  • Test publishing access before a deadline.
  • Remove outdated users after the transition is complete.
  • Document where final files and exports should be stored.

How to Maintain This Directory

Treat platform ownership as living operational information.

  • Update ownership whenever a responsibility changes.
  • Add a platform only when its purpose and owner are clear.
  • Mark tools that are discontinued or being evaluated.
  • Review the directory during the final knowledge-transfer session.
